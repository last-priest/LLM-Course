# 方向02：图形图像及多模态综合作业

## 题目描述
实现一个多模态视觉-语言助手，支持图像/视频输入的复杂推理任务（如图像描述、视觉问答、图表理解或医疗影像报告生成）。

## 题目要求
- **数据准备**：收集多模态数据集（如LLaVA-Instruct-150K、ChartQA、DocVQA或医疗影像数据集），进行清洗与对齐。
- **模型选择与微调**：基于开源多模态模型（如LLaVA-1.6、Qwen-VL、Phi-3.5-Vision）继续预训练或指令微调。
- **核心功能**：支持高清图像、多图输入、视频帧采样、复杂推理（如数学图表求解）。
- **迭代优化**：可更新投影层、LoRA秩、训练数据，提升在MMMU、ChartQA等基准上的性能。
- **部署**：构建交互Demo，支持上传图像/视频并实时回答。
- **评估**：使用GPT-4o或其他VLM作为评测器，或标准基准分数。
